Colour Game

A game to support children’s development on colour pairing.

This project involved utilising HTML5, CSS3, JavaScript and jQuery to create a game. As an individual who has a keen interest in child development, the game focused on improving children’s visual memory and processing skills using a colour sorting task. Numerical points were given depending on how quickly children registered the colour and selected the corresponding key press. To begin the planning process, I started by writing pseudo code which enabled me to understand how the code should be written. Following this, I wrote the logic for the project and completed the online game within seven days. This project idea will also enable me to create further games that support a child’s development.

Divvy

A restful application to like and share videos on the internet.

This project involved creating a full-stack restful application. I used Express to create the Front End and used Node.js to create the Back End. I also used models to create data input types. User authentication was also used to log in and out of the website. The concept of the application was focused around the recent increase in social media pertaining to video sharing. Users created an account and uploaded videos to their own personal page which then could be shared amongst other users. The planning and development of this project involved using Trello and Balsamiq wireframes. Trello was used to organise different aspects of the project and wireframes were there to help with visualising how the application would appear. This application was further developed using the Bootstrap framework.

WhatsUrBeef

A restful application that recreates traditional non-vegetarian recipes to vegan recipes.

This was a four-person group project which involved creating a full-stack application utilising AngularJS for the Front End and Node.js for the Back End. Each group member put forward their ideas and my idea was selected using a voting system. Following approval from the instructors, tasks were allocated to each group member. However, it quickly became apparent that there were limited expertise pertaining to AngularJS. Thus, we worked collectively as a team for each aspect to facilitate joint learning. This was a helpful way of working as it allowed us to learn from each other and build on strengths as well as expanding our knowledge and skills. During this project, we collaborated using Git hub via terminal on our MacBook. We learnt how to branch out development tasks rather than use one master line.

Manimal

A restful application for an online men’s store.

This project involved creating a full-stack application utilising AngularJS for the Front End and Ruby on Rails for the Back End. I chose to create an online store for men in an attempt to increase the number of online men’s stores and bridge the gap that currently exists within the consumer market. I used Trello during the planning stage and used pseudo coding to write out the logic for the code. Following this, I used Figma to create the wireframes to design the application. The framework I chose to develop the application was Tachyons. This was a helpful framework as it allowed me to input more detail with limited code. I used the SQL to build relationships between the databases and created the models in the Back End to develop the API. I used Adobe Photoshop to produce animated images throughout the application.
